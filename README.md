# vtal-fitness-tracker
VTal is a wearable fitness tracker that utilises MAX30102, BMP390, and LSM6DSOX sensors to monitor a user's physical activity and vital signs. Powered by an STM32 (NUCLEO-L432KC) microcontroller, the device streams real-time biometric and kinematic data via an HC-05 Bluetooth module to a custom PC dashboard, which utilises Qt6 for visualisation.
